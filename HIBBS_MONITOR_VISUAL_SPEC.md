## COLOR & TYPOGRAPHY HIERARCHY

### **Typography Hierarchy (Largest to Smallest)**
```
1. MONITOR (68.6pt)              ← Main title, Rubik-SemiBold
   ↓
2. THE HIBBS / Tagline (25.8pt)  ← Section headers, OpenSans-Bold
   ↓
3. Date / Section Head (21.3pt)  ← Labels, OpenSans-Light/Bold
   ↓
4. Subsection Title (13.3pt)     ← Subsection headers, OpenSans-Bold
   ↓
5. KPI Values (14pt)             ← Metric numbers, Cambria-Bold
   ↓
6. Body Text (12pt)              ← Descriptions, Cambria
   ↓
7. Labels / Body (11pt)          ← Small labels, Cambria
   ↓
8. Source / Notes (7-8pt)        ← Citations, Cambria-Italic
   ↓
9. Smallest Text (6pt)           ← Metadata, Cambria
```

### **Color Scheme**
```
Primary:
  Text: #000000 (Black)
  Background: #FFFFFF (White)
  
Secondary:
  Borders: #CCCCCC (Light Gray)
  Card BG: #F9F9F9 (Off-white)
  
Charts:
  Variable colors (Blues, Oranges, Grays)
  High contrast for accessibility

Print:
  Uses CMYK color space
  100% black text for crisp print
```

## FONT WEIGHTS & STYLES MATRIX

```
OpenSans:
  Light (300)    → Date label (21.3pt)
  Regular (400)  → Body text alternative
  SemiBold (600) → Not used in original
  Bold (700)     → Headers (13-26pt)

Rubik:
  Regular (400)  → Not used in original
  Medium (500)   → Not used in original
  SemiBold (600) → Main title (68.6pt)
  Bold (700)     → Not used in original

Cambria:
  Regular (400)  → Body text (11-12pt)
  Italic (400)   → Notes/citations (7-8pt)
  Bold (700)     → Metric values (12-14pt)
  Bold Italic    → Not used
```
