## METRIC DISPLAY FORMATS

### **Format 1: Large Number + Chart**
```
[Large Number] (e.g., "4.0%")
    ↓
[Chart visualization]
    ↓
[Description text - 50-150 words]
    ↓
[Notes line - small italic]
    ↓
[Source line - small text]
```

### **Format 2: KPI Card**
```
┌─────────────┐
│  [Number]   │  (Large, bold)
│  [Label]    │  (Small, uppercase)
│  [Date]     │  (Tiny)
└─────────────┘
```

### **Format 3: Comparison Pair**
```
[Card] [Card]   (Side by side)
[Description below]
[Source/Notes]
```

### **Format 4: Category List**
```
Utility Taxes ................. 2.6%
Natural Gas ................... 3.5%
Oil Production ................ [value]
(Each on separate line, values right-aligned)
```

## COMMON ISSUES & FIXES

| Issue | Solution |
|-------|----------|
| Fonts not loading | Add system font fallbacks; check Google Fonts CDN |
| Images too large | Set `max-width: 100%` and `height: auto` |
| Page breaks wrong place | Use `page-break-inside: avoid` on sections |
| Text wrapping wrong | Check line-height and letter-spacing |
| Colors off in print | Ensure sufficient contrast; test in print preview |
| Size too large for page | Check margins and padding; verify page dimensions |
| Charts misaligned | Ensure container is properly sized and centered |
